Configure
==========

Create .env file in ./smartcontract :

```
API_KEY= # polygonscan.com API KEY
POLYGON_MUMBAI= 
PRIVATE_KEY= # Wallet private key to deploy smart contract
```


Create .env(.local) file in ./src :

```
VITE_MORALIS_APPID=
VITE_MORALIS_SERVERURL=
VITE_SMARTCONTRACT_ADDRESS=
```
